## MLflow Server with PostgreSQL and MinIO

[GitHub Repository](https://github.com/igornunespatricio/mlflow-postgres-minio-tracking-server)

This project sets up an MLflow tracking server using Docker Compose, connected to PostgreSQL for metadata and MinIO for artifacts. Includes scripts for setup, testing, and model training/prediction pipeline.

**Key Technologies:** Docker Compose, PostgreSQL, MinIO, MLflow, Python, Bash.

## Soccer Match Forecast

[GitHub Repository](https://github.com/igornunespatricio/soccer-match-forecast)

A machine learning pipeline that predicts soccer match outcomes using web-scraped data and displays results through an interactive Streamlit dashboard. Automates data collection, transformation, model training, prediction, and dashboard display.

**Key Technologies:** Python, SQLite, Neural Networks, Selenium, BeautifulSoup, TensorFlow/Keras, Streamlit, Pandas, NumPy.
